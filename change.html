Making changes via Github

Staging and Committing

Git itself does not include any files that are moved , changed or committed in order to prevent unwanted errors.

In order to do these things there are several steps you need to follow

  • Add and commit local changes using "git add "file name or path"
  • Confirm the changes have been added to staging using "git status"
  • Commit the staged files using "git commit -m "your comment here (keep it simple and precise)"
